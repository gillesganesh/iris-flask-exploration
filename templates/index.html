Bienvenue dans l'application de prédiction des fleurs

Ce projet utilise le célèbre dataset **Iris** pour prédire le type de fleur en fonction de certaines mesures. Les variables mesurées sont les suivantes :

  • Longueur du sépale
  • Largeur du sépale
  • Longueur du pétale
  • Largeur du pétale

En envoyant ces mesures via l'API de l'application, notre modèle de machine learning vous donnera la prédiction du type de fleur (Setosa, Versicolor ou Virginica).

Comment ça fonctionne ?

Vous pouvez envoyer une requête avec les dimensions du sépale et du pétale via un appel **POST** à l'API. Le modèle vous renverra une prédiction sur le type de fleur. L'API est construite avec Flask et utilise un modèle de **Random Forest** pour effectuer la classification.

Tester l'API :

  • Utilisez l'outil **Postman** pour envoyer une requête **POST** avec les dimensions du sépale et du pétale.
  • Vous recevrez la classe prédite pour votre fleur : "Setosa", "Versicolor" ou "Virginica".
